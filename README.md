# Sistema para Gerenciamento de Campeonatos de Robótica

## RobotScore
O RobotScore é uma aplicação WEB, para o gerenciamento de campeonatos de robótica.

## 1. Introdução

### 1.1 Propósito
Este documento descreve os requisitos do sistema **RobotScore**, com objetivo de:

- Definir funcionalidades da aplicação
- Padronizar o entendimento de stakeholders
- Servir como base para desenvolvimento e testes

### 1.2 Escopo
O sistema permitirá:

- Cadastro e login de técnicos ou organizadores
- Cadastro de categorias
- Cadastro de times
- Cadstro de partidas
- Visualização de categorias, times e patidas
- Visualização de aba para premiação/colocação

O sistema será uma aplicação WEB desenvolvida com:

- **PHP** – para processar dados do servidor e regras do sistema.
- **HTML** – para estruturar as páginas do site.
- **CSS** – para controlar o visual e o layout das páginas.
- **JS** – para deixar a interface interativa e dinâmica.
- **Banco de Dados** – para guardar informações de usuários, times e partidas.

## 1.3 Definições

| Termo | Definições |
| - | - |
| Organizadores | Cada categoria possui um responsável pelo campeonato localmente |
| Técnicos | Cada equipe possui no mínimo um técnico em sua formação para auxílio |
| Categorias | Modalide de robótica com sesus próprio objetivos e regras |
| Times | Equipes dentro de categorias destinadas a desenvolver seus projetos |
| Partidas | Parte prática onde o robô entra em ação (formato varia conforme categoria) |
| Premiação | Seção do campeonato destinada à congratulação dos melhores colocados |
